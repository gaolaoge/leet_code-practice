二叉搜索树（BST）是一种二叉树，满足以下性质：

• 左子树所有节点值 < 根节点值  
 • 右子树所有节点值 > 根节点值  
 • 左右子树也是二叉搜索树

支持高效查找、插入、删除操作，时间复杂度平均O(log n)，最坏O(n)（退化成链表）。
