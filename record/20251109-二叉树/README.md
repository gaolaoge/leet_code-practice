# 二叉树

## 定义

二叉树是一种树形数据结构，其中每个节点最多有两个子节点，通常称为左子节点和右子节点。

## 基本特点

- **节点结构**：每个节点包含三部分：数据域、左子树指针、右子树指针
- **层级关系**：有明确的父子关系和层级结构
- **递归性质**：二叉树的子树本身也是二叉树

## 常见类型

1. **满二叉树**：每个节点要么是叶子节点，要么有两个子节点
2. **完全二叉树**：除最后一层外，其他层的节点都是满的，最后一层节点从左到右连续
3. **二叉搜索树（BST）**：左子树所有节点值 < 根节点值 < 右子树所有节点值
4. **平衡二叉树（AVL）**：任意节点的左右子树高度差不超过1
5. **红黑树**：自平衡的二叉搜索树

## 遍历方式

- **前序遍历**：根 → 左 → 右
- **中序遍历**：左 → 根 → 右（BST中可得到有序序列）
- **后序遍历**：左 → 右 → 根
- **层序遍历**：逐层从左到右遍历（使用队列实现）

## 常见操作

- 插入节点
- 删除节点
- 查找节点
- 计算高度/深度
- 判断平衡性
- 翻转二叉树
- 寻找最近公共祖先

## 应用场景

- 表达式解析（表达式树）
- 文件系统（目录结构）
- 数据库索引（B树、B+树）
- 编译器（语法树）
- 决策算法（决策树）
- 霍夫曼编码（压缩算法）

## 时间复杂度

- 查找/插入/删除：
  - 平衡二叉树：O(log n)
  - 最坏情况（退化为链表）：O(n)
