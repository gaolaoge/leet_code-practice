| 数据结构       | 时间复杂度                 | 空间复杂度               | 核心优势               | 典型应用场景         |
| -------------- | -------------------------- | ------------------------ | ---------------------- | -------------------- |
| **数组**       | 查找O(1)<br>插入/删除O(n)  | O(n)                     | 随机访问快             | 缓存、DP、矩阵运算   |
| **链表**       | 查找O(n)<br>插入/删除O(1)  | O(n)                     | 动态大小<br>插入删除快 | LRU缓存、链式结构    |
| **栈**         | 所有操作O(1)               | O(n)                     | LIFO特性               | 函数调用、表达式求值 |
| **队列**       | 所有操作O(1)               | O(n)                     | FIFO特性               | 广度搜索、任务调度   |
| **双端队列**   | 所有操作O(1)               | O(n)                     | 两端操作               | 滑动窗口、回文检查   |
| **哈希表**     | 查找/插入/删除O(1)         | O(n)                     | 快速查找               | 缓存、计数、去重     |
| **二叉树**     | 查找O(n)<br>遍历O(n)       | O(n)                     | 层次结构               | 文件系统、语法树     |
| **二叉搜索树** | 查找/插入/删除O(log n)     | O(n)                     | 有序性<br>范围查询     | 数据库索引、排序     |
| **平衡BST**    | 所有操作O(log n)           | O(n)                     | 稳定性能<br>有序性     | 标准库map/set、索引  |
| **堆**         | 插入O(log n)<br>取极值O(1) | O(n)                     | 极值快速获取           | 优先队列、堆排序     |
| **并查集**     | 查找/合并O(α(n))           | O(n)                     | 连通性判断             | 图连通、Kruskal算法  |
| **字典树**     | 查找O(m)<br>插入O(m)       | O(ALPHABET_SIZE × N × M) | 前缀匹配               | 自动补全、IP路由     |
| **线段树**     | 查询/更新O(log n)          | O(n)                     | 区间操作               | 区间最值、区间和     |
| **树状数组**   | 查询/更新O(log n)          | O(n)                     | 区间前缀和             | 离散化、计数问题     |
| **图**         | 取决于表示方法             | O(V+E)                   | 关系建模               | 社交网络、路径规划   |

| 解题思路     | 核心思想                   | 适用场景            | 时间复杂度             | 典型应用                 |
| ------------ | -------------------------- | ------------------- | ---------------------- | ------------------------ |
| **线性遍历** | 逐一访问每个元素           | 数组、链表基础操作  | O(n)                   | 查找、求和、过滤         |
| **二分查找** | 折半缩小搜索范围           | 有序数据查找        | O(log n)               | 有序数组搜索、旋转数组   |
| **递归**     | 问题分解为相同类型的子问题 | 树操作、分治算法    | 取决于问题             | 树遍历、斐波那契         |
| **分治法**   | 分解→解决→合并             | 可分解的复杂问题    | 取决于问题             | 归并排序、快速幂         |
| **动态规划** | 记忆化递归，状态转移       | 最优子结构问题      | O(状态数×转移数)       | 背包、最长子序列         |
| **贪心算法** | 每步局部最优               | 局部最优推全局最优  | O(n)                   | 活动选择、分数背包       |
| **回溯算法** | 试错，不满足时回退         | 约束满足问题        | 指数级                 | N皇后、组合排列          |
| **双指针**   | 两个指针协同工作           | 数组、链表操作      | O(n)                   | 两数之和、反转数组       |
| **滑动窗口** | 维护固定/可变窗口          | 子数组/子串问题     | O(n)                   | 最小覆盖子串、最长无重复 |
| **快慢指针** | 不同速度的指针检测         | 链表环、去重问题    | O(n)                   | 环形链表、删除重复       |
| **前缀和**   | 预处理区间和               | 区间查询问题        | 预处理O(n)<br>查询O(1) | 区间和、子数组和         |
| **单调栈**   | 维护单调性解决特定问题     | 下一个更大/更小元素 | O(n)                   | 括号匹配、柱状图         |
| **拓扑排序** | 依赖关系排序               | 有向无环图          | O(V+E)                 | 课程表、依赖解析         |
| ** BFS**     | 层次遍历，队列实现         | 最短路径、层次问题  | O(V+E)                 | 最短路径、层序遍历       |
| ** DFS**     | 深度优先搜索               | 连通性、路径问题    | O(V+E)                 | 岛屿数量、路径查找       |
| **并查集**   | 处理不相交集合合并查找     | 连通性问题          | O(α(n))                | 岛屿合并、Kruskal        |
